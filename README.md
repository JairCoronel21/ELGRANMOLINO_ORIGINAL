# Modificar
En caso de querer cambiar algo. Primero crear una nueva rama

## Crear rama y cambiarse
- `git branch <rama>`
- `git checkout <rama>`

## Guardar cambios y pushear rama
- `git add .`
- `git commit -m "<mensaje>"`
- `git push -u origin <rama>`
